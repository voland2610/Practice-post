# Practice Post

Простое React-приложение для работы с API JSONPlaceholder.  
В проекте реализованы: пользователи, посты и комментарии с переключением вкладок (tabs), а также отображение комментариев к постам.

## 🚀 Стек технологий

- React 18  
- Vite  
- React Router DOM  
- Fetch API для запросов к JSONPlaceholder  

## 🖥️ Структура приложения

- `/` – Приветственная страница (WelcomePage) с кнопкой «Начать»  
- `/content` – Основная страница с тремя вкладками:
  - **Пользователи** – отображает имя, email, телефон и компанию  
  - **Посты** – отображает title и body поста; при клике на пост показываются комментарии  
  - **Комментарии** – отображает список всех комментариев с name, email и body  

## ⚡ Установка

1. Клонируйте репозиторий:
git clone https://github.com/voland2610/Practice-post.git

2. Перейдите в папку проекта:
cd Practice-post

3. Установите зависимости:
npm install

4. Запуск
npm run dev
